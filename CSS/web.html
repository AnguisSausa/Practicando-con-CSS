La web, también conocida como World Wide Web (WWW), es un sistema global de información basado en internet que permite a las personas acceder, compartir y interactuar con contenido digital. Fue inventada en 1989 por Tim Berners-Lee, un científico británico del CERN, como una manera de compartir información entre investigadores de una forma más eficiente. Con el tiempo, se ha convertido en uno de los pilares fundamentales de la comunicación y el intercambio de información en el mundo moderno. El funcionamiento de la web se basa en una estructura cliente-servidor. Esto significa que los dispositivos que acceden a la web, como computadoras, teléfonos inteligentes o tablets (los clientes), envían solicitudes para acceder a recursos específicos (como páginas web) que están almacenados en computadoras remotas llamadas servidores. Los servidores responden enviando los datos solicitados al cliente, que luego los muestra a través de un navegador web, como Google Chrome, Mozilla Firefox o Microsoft Edge. El contenido en la web está compuesto por páginas web, las cuales se escriben en lenguajes como HTML (HyperText Markup Language) y se pueden enriquecer con CSS (Cascading Style Sheets) para diseño y JavaScript para interactividad. Estas páginas están enlazadas entre sí mediante hipervínculos, creando una vasta red de información interconectada. El acceso a la web se realiza a través de internet, que es la infraestructura física y lógica que conecta millones de redes y dispositivos en todo el mundo. La web, por lo tanto, es un servicio que opera sobre internet, junto con otros servicios como el correo electrónico, la transmisión de archivos y las redes sociales. Además, la web puede dividirse en diferentes niveles según su accesibilidad y el tipo de contenido que almacena: La Web Superficial: Es la parte de la web que está indexada por motores de búsqueda como Google y a la que la mayoría de las personas accede. La Web Profunda: Contiene contenido que no está indexado por motores de búsqueda, como bases de datos privadas, correos electrónicos y contenido detrás de inicios de sesión. La Web Oscura: Una sección más pequeña que requiere software especial como Tor para acceder y a menudo asociada con actividades anónimas. Hoy en día, la web es una herramienta esencial para la educación, el entretenimiento, los negocios y la comunicación global.
